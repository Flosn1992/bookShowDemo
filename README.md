##项目描述
这个是一个可以展示书信息的东西，能够对书进行录入，修改，删除并可以列表展示。

##项目注意
这个东西，是前端的，所以这里的功能测试我准备了一些加注释

##项目的所依赖技术
初步：Vue + Element UI + Axios
初步：Sprintpool
数据库：Idea自带的数据库
